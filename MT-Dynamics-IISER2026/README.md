# MT-Dynamics-IISER2026

Microtubule dynamics analysis and modeling.

## Project Structure
- `data/`: Raw and processed data.
- `src/`: Source code for preprocessing, segmentation, tracking, and modeling.
- `notebooks/`: Jupyter notebooks for exploration and analysis.
- `dashboard/`: Streamlit dashboard for visualizing results.
- `results/`: Output files and figures.
- `report/`: Documentation and project report.
